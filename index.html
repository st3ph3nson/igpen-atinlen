Nordic Drift Translator

Applies a controlled phonetic drift to English text (case-preserving, punctuation-safe).

Rules: W→V, TH→D, I↔E, A↔O, Y↔U, H removed between nouns
Example: I am driving to the store with my son. → E om dreveng ta di stari ved mu san.
Note: “H removed between nouns” uses a lightweight heuristic (capitalized/proper nouns and common noun patterns). You can tune it in the code if you want it stricter or looser.
