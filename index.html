-eg Translator

Only words that start with a vowel (A/E/I/O/U/Y) change: last letter → front + eg. Consonant-start words stay exactly the same.

Examples: open → nopeeg, always → salwayeg, either → reitheeg, and → daneg, a → aeg, an → naeg, you → uoyeg
Rule: vowel-start (AEIOUY) length ≥ 1 → rotate-last-to-front + eg. Consonant-start words pass through unchanged.
