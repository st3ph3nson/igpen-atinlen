EN Pig Latin Translator

Rules (selected ending applies): Consonant-start words move the starting consonant cluster to the end + ending (the → e-th + ending). Vowel-start words: if ending is -yay, add “yay” (apple → appleyay); otherwise add the ending (are → areen / areeg). “I” / “is” keep the base casing but always attach the ending: I → Ien, Is → Isen, is → isen. Punctuation and spacing are preserved. Decode is best-effort (Pig Latin is inherently ambiguous).
