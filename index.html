-an Translator

Consonant-start → Pig Latin (consonant cluster) + an. Vowel-start (A/E/I/O/U/Y) → last letter to front + an.

Examples: open → nopean, you → uoyan, Friday → idayfran, street → eetstran
Rules: vowel-start (AEIOUY) length≥2 → rotate-last-to-front + an. Otherwise → move initial consonant cluster + an.
